STEP 4: Record Account Balance
=======================================================================

1. Navigate back to the main Dashboard (click "Dashboard" or the OA logo).
2. Locate the account balance display (usually top card or sidebar).
3. SCREENSHOT: dashboard showing balance.

4. READ the balance and report it to Aryan in your summary.

5. Calculate the allocation scaling factor:
   - factor = current_balance / 50000
   - If current_balance = $50,000 exactly → factor = 1.0
   - If current_balance = $25,000 → factor = 0.5
   - If current_balance = $100,000 → factor = 2.0

6. Include this factor in your final summary — later tasks use it to
   scale per-bot allocations from the $50k baseline.
