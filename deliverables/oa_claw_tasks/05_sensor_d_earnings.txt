CONTEXT
=======================================================================

Sensor D tags bots with CLUSTER_RISK when any of the top-5 QQQ holdings
has earnings in the next 3 trading days. Only Bot 7 (qqq-ic-extension)
uses this tag — QQQ is ~60% mega-cap concentrated (NVDA, MSFT, META,
GOOGL, AMZN), and their earnings create outsized gap risk that the
regime classifier doesn't capture.

Simplified build: tag CLUSTER_RISK if ANY of the 5 mega-caps has
earnings within 3 days. More conservative than requiring a true cluster
but much easier to build in OA.
