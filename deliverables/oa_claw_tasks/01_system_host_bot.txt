=======================================================================
CONTEXT
=======================================================================

This bot will NEVER trade. It exists solely as a container to host four
shared sensor automations (Sensors A, B, C, D) that tag every trading bot
with the current market regime each morning.

OA requires a minimum bot allocation of $1,000 even for bots that don't
trade. We satisfy the minimum while setting position limits to 0 so no
actual trading can happen from this bot.

=======================================================================
SAFETY RULES
=======================================================================

1. PAPER mode only (should have been verified in Task 00)
2. Position limits MUST be 0 — if you accidentally set them > 0 and the
   bot runs a decision tree that places an order, that's a bug
3. Screenshot before and after the Create Bot button click
4. If any field in the bot creation form differs from below, STOP

=======================================================================
STEP 1: Navigate to Bot Creation
=======================================================================

1. From the OA Dashboard, click "Bots" in the main navigation.
2. Click "+ New Bot" button (usually top-right of the bots page).
3. SCREENSHOT: the bot creation dialog/form.

If the button is labeled differently (e.g., "Create Bot", "Add Bot"),
that's fine — click it. If there's NO button to create a new bot, STOP.

=======================================================================
STEP 2: Fill in Bot Creation Form
=======================================================================

Enter these EXACT values (refer to Worksheet 1 for the canonical list):

Field: Name
Value: SYSTEM - Regime Sensor Host

Field: Symbols (or "Underlying Symbol" / "Watchlist")
Value: SPY
NOTE: Even though this bot doesn't trade, OA requires at least one
symbol. Use SPY as a placeholder.

Field: Allocation (or "Capital Allocated")
Value: $1,000
NOTE: This is the OA minimum. Do NOT set higher.

Field: Total open positions (or "Max open positions")
Value: 0
CRITICAL: Must be 0. This prevents any order from being placed.

Field: Daily open positions (or "Max positions per day")
Value: 0
CRITICAL: Must be 0.

Field: Broker
Value: Use whatever broker was confirmed connected in Task 00

Field: SmartPricing
Value: Normal (default)

Field: Scan Speed
Value: Standard (15 min, default)

Field: Auto-disable bot
Value: No (or unchecked)

Field: PDT Protection
Value: Yes / checkbox ON

SCREENSHOT: the completed form before clicking Create.

PAUSE POINT 1: Review the form. Are all position limits set to 0?
Is allocation $1,000? If yes, proceed. If any field shows a different
value than above, STOP and show Aryan.
